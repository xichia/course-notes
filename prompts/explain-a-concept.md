# Explain a Concept

## Prompt

Use `manifest.json` to find **[concept or note ID]** in **[course]**. Open its concept note plus only the prerequisites needed for this explanation.

Explain it at **[introductory/intermediate/advanced]** level. Start with the note's precise definition, then give intuition, connect it to prerequisites, and walk through one existing example. Explicitly separate:

- what the repository says;
- any external background you add;
- gaps or ambiguities in the notes.

Finish with one check-for-understanding question. Cite the note IDs or file paths used. Do not fill missing course-specific content by guessing.

## Publication Safety

Treat course-derived explanations as private by default. Preserve `visibility` and `source-risk`. If explicitly asked for public-facing material, remove protected course expression and prefer an independently written generic explanation with synthetic examples.
